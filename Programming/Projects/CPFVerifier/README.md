# CPF Verifier (Verificador de CPF)
Projeto apresentado como requisito na disciplina de Algorítmos e Programação II do curso de Bacharelado em Sistemas 
da Informação, do Instituto Federal Catarinense - Campus Araquari. 

### Teste de CPF
O arquivo "funcoesCPF.py" conta com os métodos usados nas demais funções dentro do projeto. Esses métodos foram 
desenvolvidos e testados com o uso do "assert" do python.

### Validação de CPF
O arquivo "validaCPF.py" abre uma interface gráfica construída com a ferramenta Tkinter, com a função de receber
um valor de CPF do usuário e devolver se o mesmo é válido ou não.
